레이어 책임​

Stage레이어폴더책임
S0Datadata/원천 데이터 수집, 정제, 품질 검증
S1Universedata/투자 가능 종목 필터링
S2Signalssignals/팩터/이벤트 시그널 생성
S3Screenerselection/1차 필터링 (Hard Cut)
S4Rankingselection/종합 점수 산출
S5Portfolioportfolio/포트폴리오 구성
S6Executionexecution/주문 생성/체결
S7Auditaudit/성과 분석

핵심 원칙​

1. 단방향 데이터 흐름​

S0 → S1 → S2 → S3 → S4 → S5 → S6
                                 ↓
                                S7 (피드백)

2. Contract 기반 통신​

각 레이어는 정해진 타입으로만 데이터 주고받음

3. Brain = Orchestrator Only​

Brain은 로직 없이 파이프라인 실행 순서만 관리
